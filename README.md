Tasbih & Zikir - by QayyumAmin

A beautiful, feature-rich Islamic Tasbih counter and Zikir reminder app built as a Progressive Web App (PWA). Works on any browser and can be installed directly to your iPhone or Android home screen.

Features

📿 Count Tasbih with haptic feedback and completion animations

🔥 Daily streak tracker with 7-day history

🌙 Preset Zikir packs Tasbih, Morning Adhkar, Salawat, Istighfar

🔔 Zikir reminders with custom days and times

📊 Stats dashboard with session history and weekly chart

🌐 3 languages English, Bahasa Melayu, Bahasa Indonesia

💾 Backup and restore your data (JSON export/import)

📱 PWA, install to iPhone and Android home screen with custom icon

🕌 Works fully offline
